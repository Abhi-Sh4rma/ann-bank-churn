# 💳 Bank Customer Churn Prediction using ANN

## 📌 Overview
This project predicts whether a bank customer will leave (churn) using an Artificial Neural Network (ANN).

## 🧠 Model Details
- Input features: Customer data (credit score, geography, gender, etc.)
- ANN with multiple hidden layers
- Activation: ReLU & Sigmoid
- Optimizer: Adam

## 📊 Results
- Accuracy: ~85–88%
- Improved performance using feature scaling and model tuning

## ⚙️ Technologies Used
- Python
- NumPy
- Pandas
- Scikit-learn
- TensorFlow / Keras
